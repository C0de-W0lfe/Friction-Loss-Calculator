Friction Loss Calculator

Calculate head loss in pipes (in feet) using Hazen-Williams formula

Typical C values: 120–150 for new steel/iron pipes, 100–120 for older pipes, or 140 for smooth pipes like plastic. Formula: Hf = 10.44 * (Q/C)^1.85 * L / D^4.87, where Q is in GPM, L in feet, D in inches, and C is dimensionless.
